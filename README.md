# HR Analytics Dashboards in Excel: Workforce, Retention & Compensation Insights

## Overview  
This project focuses on designing and developing interactive HR dashboards in Excel to improve visibility into workforce performance, employee retention, and compensation efficiency.

It includes three dashboards tailored for different stakeholders, enabling data-driven decision-making across the organization.

---

## Dashboard Overview  

### Executive Dashboard  
Provides a high-level summary of workforce performance for senior leadership, focusing on productivity, attrition, and workforce demographics.

![Executive Dashboard](images/executive_dashboard.png)

---

### Retention Dashboard  
Analyzes employee attrition trends and highlights key factors influencing employee turnover and retention effectiveness.

![Retention Dashboard](images/retention_dashboard.png)

---

### Compensation Dashboard  
Focuses on salary distribution, compensation growth, and cost efficiency to support financial planning and control.

![Compensation Dashboard](images/compensation_dashboard.png)

---

## Key Features  
- KPI-driven dashboards for different business functions  
- Data cleaning and anomaly handling for accurate insights  
- Interactive Excel visuals using slicers and pivot tables  
- Business-focused metrics for decision-making  

---

## Tools and Technologies

- Pivot Tables and Charts  
- Data Cleaning and Transformation  
- Dashboard Design
